## Especificaciones Técnicas

### Formatos Recomendados
- **Logos**: PNG con fondo transparente
- **Fotos**: JPG optimizado
- **Iconos**: PNG o SVG

### Dimensiones Sugeridas
- **Logo header**: 300x120px (50% más presencia visual)
- **Logo footer**: 360x145px (proporcional al header)
- **Hero image**: 800x600px
- **Service icons**: 100x100px
- **Client logos**: 200x100px (formato horizontal recomendado)

### Optimización
- Comprimir imágenes antes de usar
- Usar herramientas como TinyPNG o ImageOptim
- Mantener calidad pero reducir tamaño de archivo

## Nota Importante

⚠️ **Las imágenes no están incluidas en el template por derechos de autor.**

Debes crear o obtener tus propias imágenes para:
1. Logos de tu empresa
2. Fotos del equipo y oficinas
3. Iconos de servicios
4. Testimonios reales de clientes
